My Todo App

A simple and efficient Todo application built with React and Zustand for state management. This project is my first hands-on build after learning Front-End Development, showcasing how to manage tasks with add, toggle, and delete functionality.



 Features
 
Add new todos with a title and description

Toggle completion status (mark tasks as done/undone)

Delete tasks with a single click

Clean and responsive UI

State management powered by Zustand
